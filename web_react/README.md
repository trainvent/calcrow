# CSVrow

Smart CSV row editor focused on monthly work logs.

## What's included
- CSV parsing + date format detection
- Monthly table generator with summary row
- Widget-based row editor (time pickers, slider, diary)
- Download updated CSV

## Next steps (planned)
- Google Drive connector
- URL loading + save-back flow
- Supabase Auth (Google + email)
- Persistent local cache

## Run
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
